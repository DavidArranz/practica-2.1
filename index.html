Navegadores web

Un navegador web (in inglés, web browser)es un sogrware, aplicación o programa que permite el acceso a la Web, interpretando la informacion de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualización de documentos de texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadors web

Safari:safari es un navegador web de código cerrado desarrollado por apple inc. Está disponible para macOS e iOS.

Mozilla Firefox es un navegador web libre y se código abierto desarrollado para Linux, Android, iOS, macOS y Microsogr Windows coordinado por la fundación Mozilla.

Googlechrome es un navegador web de código cerrado desarrollado por google, aunque derivado de proyectos de código abierto
